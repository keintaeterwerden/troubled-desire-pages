گلابی رنگ کے شیشے

ذہنی تعصب کیلئے مثال کے طور پر گلابی رنگ کے شیشے

تصور کریں کہ آپ کسی سے محبت کرتے ہیں۔ اس خاص مرحلے میں آپ دنیا کو بہت زیادہ پُر امید نقطہ نظر کے ساتھ، یا "گلابی رنگ کے شیشوں" کے ذریعے دیکھ سکتے ہیں۔ خواہش کی چیز سے آنے والے تمام اشاروں اور نظروں کی تاویل دلچسپ، قصدی اور تصدیقی طور پر کی جاتی ہے حالانکہ ان "چشموں" کے بغیر کوئی شخص ان ہی افعال کا بالکل مختلف انداز میں جائزہ لے سکتا ہے۔ واضح اور معروضی طرز نظر کو دوبارہ حاصل کرنے کیلئے اس سے آگاہ ہونا، اور مسخ کرنے والے شیشوں کو "اُتار" دینا ضروری ہے۔

جنسی حملے کے سلسلے میں ذہنی تعصبات مجرم سے متاثرہ کی طرف ذمہ داری منتقل کرنے کا سبب بن سکتے ہیں۔ اگر کوئی شخص کسی بچے کے ساتھ جنسی بدسلوکی کرتا ہے، تو وہ ذہنی تعصبات حملہ سے پہلے، دوران اور بعد میں فعال طور پر اثر انداز ہو رہے ہوتے ہیں۔ درج ذیل بیانات اس بات کو ثابت کر سکتے ہیں: "بچہ اپنی جنسیت کا تجربہ اسی طرح کرنا چاہتا ہے جیسا کہ بالغین کرتے ہیں،" "اگر بچہ مزاحمت نہیں کرتا ہے، تو وہ اسے ضرور چاہے گا،" اور "یہ آخری بار تھا۔" اگر آپ ان ذہنی تعصبات کو پہچاننے اور ان میں ترمیم کرنے میں کامیاب نہیں ہوتے ہیں، تو آپ (دوبارہ) جنسی حملہ کرنے کا نقصان اٹھاتے ہیں۔

آپ اپنے گلابی رنگ کے شیشوں کو کیسے ایڈجسٹ کر سکتے ہیں؟

ایسی صورتحال کے بارے میں سوچیں جس میں آپ کو حملہ کرنے کا زیادہ خطرہ تھا یا جس میں آپ جنسی حملہ کر چکے تھے۔ پھر اپنا جائزہ لینے کیلئے درج ذیل خاکہ استعمال کریں:

ذہنی تعصب

  • کو ظاہر کریں آپ کے دماغ میں کیا چل رہا تھا؟
  • اس صورتحال میں آپ کی عین سوچ کیا تھی؟
  • اس صورتحال میں آپ نے اپنے آپ کو کیا کہا؟

ذہنی تعصب پر سوال

  • اٹھانا کیا آپ کی سوچ معقول تھی؟
  • کیا آپ کی سوچ حقائق پر مبنی تھی؟
  • کیا آپ کی سوچ کے درست ہونے کا کوئی ثبوت تھا؟
  • کیا وہ سوچ مفید تھی؟
  • اس طرح سوچنے کا کیا فائدہ ہوا؟ کیا نقصان ہوا؟

ایک مناسب عملی سوچ تیار

  • کرنا گذشتہ کا ملاحظہ کرتے ہوئے، آپ کے خیال میں اس صورتحال میں ایک اور ممکنہ سوچ کیا ہو سکتی تھی؟
  • آپ کے خیال میں آپ کا کوئی دوست اس صورتحال میں کیا سوچ رہا ہوگا؟
  • صورتحال کی کون سی تاویل زیادہ حقیقت پسندانہ لگتی ہے؟
  • اس کے بجائے آپ اپنے آپ کو کیا کہہ سکتے ہیں اگر آپ کو دوبارہ یہی صورتحال پیش آ جائے؟

عملی سوچ کی مشق

  • اپنے آپ کو اس عملی سوچ کے فوائد سے آگاہ کریں۔
  • انڈیکس کارڈ پر اپنی عملی سوچ لکھیں۔
  • اس عملی سوچ کو اپنے آپ پر بار بار دہرائیں۔
  • تصور کریں کہ اگلی بار جب آپ کسی اعلی خطرے کی صورتحال میں ہوں گے تو آپ اپنے آپ کو اس عملی سوچ کی کیسے یاد دلائیں گے
