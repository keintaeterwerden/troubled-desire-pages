ನಿಮ್ಮ ಆನ್‌ಲೈನ್ ಸೆಷನ್

ಹೊಸ ಸೆಷನ್

ದಯವಿಟ್ಟು ಇಲ್ಲಿ ಹೊಸ ಸೆಷನ್ ಪ್ರಾರಂಭಿಸಿ! ಅನಾಮಧೇಯವಾಗಿ ಮತ್ತು ರಹಸ್ಯವಾಗಿರುತ್ತದೆ.

ಹಿಂದಿನ ಸೆಶನ್ ಪ್ರವೇಶಿಸಿ.

ನಿಮ್ಮ ಸೆಷನ್ ಅನ್ನು ಮರುಭೇಟಿ ಮಾಡಲು ದಯವಿಟ್ಟು ನಿಮ್ಮ ಸೆಷನ್ ID ಯನ್ನು ನಮೂದಿಸಿ!

ಪ್ರತಿ ಬಾರಿ ನೀವು ಹೊಸ ಸೆಶನ್ ಅನ್ನು ಪ್ರಾರಂಭಿಸಿದಾಗ, ನೀವು ಸೆಷನ್ ID ಯನ್ನು ಸ್ವೀಕರಿಸುತ್ತೀರಿ. ಮೇಲಿನ ಬಲಭಾಗದಲ್ಲಿ ನೀವು ಸೆಷನ್ ID ಯನ್ನು ನೋಡಬಹುದು. ನೀವು ನಂತರ ಈ ಆಧಾರಗಳನ್ನು ಮತ್ತೆ ಬಳಸಲು ಬಯಸಿದಲ್ಲಿ ಅಥವಾ ನಂತರದ ಸಮಯದಲ್ಲಿ ಮುಂದುವರಿಸಲು ಬಯಸಿದರೆ ದಯವಿಟ್ಟು ನಿಮ್ಮ ಸೆಷನ್ ID ಯನ್ನು ಗುರುತು ಹಾಕಿಕೊಳ್ಳಿ. ನಿಮ್ಮ ID ಯನ್ನು ನೀವು ಕಳೆದುಕೊಂಡಿದ್ದರೆ ನಿಮಗೆ ಮರು-ಲಾಗಿನ್ ಮಾಡಲು ಸಾಧ್ಯವಾಗುವುದಿಲ್ಲ. ಈ ಸಂದರ್ಭದಲ್ಲಿ, ನೀವು ಸೆಷನ್ ಅನ್ನು ಪುನರಾವರ್ತಿಸಬೇಕಾಗಿರುತ್ತದೆ. ನಿಮ್ಮ ಉತ್ತರವನ್ನು ಆಧರಿಸಿ ನೀವು ಚಿಕಿತ್ಸಕರನ್ನು ಸಂಪರ್ಕಿಸಲು ಅರ್ಹರಾಗಬಹುದು. ಕಡ್ಡಾಯ ವರದಿಗೆ ಸಂಬಂಧಿಸಿದಂತೆ ನಿಮ್ಮ ಸ್ಥಳೀಯ ಕಾನೂನಿನ ಬಗ್ಗೆ ಯಾವಾಗಲೂ ತಿಳಿದಿರಲಿ. ಆನ್‌ಲೈನ್ ಚಿಕಿತ್ಸಾ ಅವಧಿಯನ್ನು ಮುಂದುವರಿಸುವುದು ಅಥವಾ ಬಿಡುವುದು ನಿಮ್ಮ ಆಯ್ಕೆಯಾಗಿರುತ್ತದೆ.
