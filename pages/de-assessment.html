Session fortsetzen

Session ID eingeben um Session fortzusetzen

Wenn Sie eine neue Session starten erhalten Sie eine Session ID. Sie sehen diese oben rechts. Bitte notieren Sie sich Ihre Session ID für den Fall, dass Sie die Session zu einem späteren Zeitpunkt fortsetzen möchten. Ein Passwort wird nicht benötigt. Wenn Sie Ihre persönliche ID verlieren können Sie nicht mehr auf Ihre begonnene Session zugreifen. In diesem Fall müssen Sie leider von vorn beginnen. Beachten Sie bitte die lokale Gesetzgebung bezüglich der Meldepflicht, wenn Sie Therapeuten kontaktieren. In Deutschland sind Sie komplett durch die therapeutische Schweigepflicht geschützt. Es gibt immer die Möglichkeit die Online-Selbsthilfe Session fortzusetzen.
