Setiap kali memulai sesi baru, Anda akan menerima ID sesi. Anda dapat melihat ID sesi di bagian kanan atas. Salin ID sesi jika Anda bermaksud menggunakan kredensial itu nanti atau ingin melanjutkan sesi pada lain waktu. Jika kehilangan ID, Anda tidak dapat login kembali. Jika hal ini terjadi, Anda harus mengulangi sesi. Berdasarkan tanggapan Anda, Anda mungkin memenuhi syarat untuk menghubungi terapis. Selalu perhatikan peraturan setempat mengenai wajib lapor. Untuk melanjutkan, selalu ada pilihan sesi perawatan online.
