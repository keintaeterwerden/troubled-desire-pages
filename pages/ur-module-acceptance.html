قبولیت کیا ہے؟

تصور کریں کہ آپ ایک بس ڈرائیور ہیں جو کسی مخصوص منزل تک جانے والی سڑک پر چل رہے ہیں۔ جس سڑک پر آپ گاڑی چلا رہے ہیں وہ گہرے گڑھوں اور ٹکڑوں والی خمیدہ سڑک ہے۔ بعض اوقات، آپ کو سڑک پر اپنی نظر برقرار رکھنے میں بھی مشکلات پیش آتی ہیں۔ راستہ چلتے آپ کو مسافروں کی ایک وسیع اقسام کو جاننے کا موقع ملتا ہے۔ کچھ بدتمیز ہوتے ہیں اور اپنی رائے کا اظہار اونچی آواز اور جارحانہ انداز میں کرتے ہیں، کچھ خاموشی سے بیٹھے رہتے ہیں اور دوسرے آپ کی طرف چیخ کر آپ کو ڈراتے ہیں۔ تصور کریں کہ یہ بس آپ کا شعور ہے اور آپ اس سارے افراتفری کے درمیان خود کو سیدھا چَلانے کی کوشش کر رہے ہیں۔ ان مسافروں میں سے ایک کو تنہائی کہا جا سکتا ہے، دوسرے کو پیڈوفیلیا یا خود بیزاری کہا جا سکتا ہے۔ یہ مہمان بلند آواز اور مستقل مزاج ہیں۔ آپ جتنا سڑک پر رہنے کی کوشش کرتے ہیں اتنا ہی وہ آپ کو پریشان کرتے ہیں۔ وہ آپ کو دائیں یا بائیں طرف مڑنے کا مشورہ دیتے ہیں یا اس کیلئے قائل کرنے کی کوشش کرتے ہیں۔ دوسرے آپ کی حوصلہ شکنی کرتے ہیں اور کہتے ہیں کہ آپ ناکام ہوں گے، چاہے آپ جو بھی کوشش کر لیں۔ تھوڑی دیر کے بعد، آپ کو احساس ہو سکتا ہے کہ آپ ان مہمانوں سے نمٹنے میں اتنے مصروف ہو گئے ہیں کہ آپ کوئی ٹریفک سائن چوک جاتے ہیں یا غلط راستے پر مڑ جاتے ہیں۔ ہو سکتا ہے کہ آپ بجائے آگے راستے پر توجہ دیںے کے، آپ بس کو روک دیں، پیچھے مڑیں اور اپنے مہمانوں کو جھڑکیں۔ آپ آگے نہیں بڑھتے ہیں۔ اس کے بجائے، آپ ان چیزوں سے نمٹنے میں مشغول ہو جاتے ہیں جن کا آپ کی منزل سے کوئی تعلق نہیں ہے۔ یہ آپ کی مرضی ہے: یا تو آپ مسافروں کے ساتھ جھگڑا اور جدوجہد جاری رکھیں، یا آپ انہیں رہنے دیں، ہینڈل پکڑیں اور سڑک اور اس منزل پر توجہ مرکوز کر لیں جس کی طرف آپ جا رہے ہیں۔ ممکنہ طور پر آپ کو پورے گروپ کو ساتھ لے کر جانا پڑے گا اور آپ ان سے چھٹکارا حاصل نہیں کر پائیں گے۔ تاہم، یہ آپ کا فیصلہ ہے کہ آپ اپنے سفر پر توجہ مرکوز کریں یا اپنے مہمانوں کے چیخنے پر۔

پھر اب کیا کریں؟

ہم پہلے ہی اس مفروضے سے واقف ہو چکے ہیں کہ جنس، جسمانی عمر اور پیش رفت کیلئے جنسی ترجیح شاید زندگی بھر مستحکم رہتی ہے۔

کیا یہ واقعی اتنا برا ہے؟

یہ کچھ لوگوں کیلئے صدمہ ہو سکتا ہے، لیکن یہ درحقیقت کوئی تباہی نہیں ہے۔

جنسی ترجیح ہماری شخصیت کا صرف ایک حصہ ہے۔ ہماری شخصیت بہت سے اجزاء پر مشتمل ہوتی ہے۔ اس میں ہماری اقدار اور اخلاقی ضابطے، ہماری طاقتیں اور کمزوریاں، ہماری شخصیت کی خصلتیں اور حقیقی برتاؤ شامل ہیں۔ یہ تمام صفات اور بہت کچھ ہماری شخصیت کو تشکیل دیتے ہیں۔ یاد رکھیں، صرف ہمارا برتاؤ ہی سماجی سیاق و سباق میں دوسروں کے ہمارے بارے میں تاثر کا تعین کرتا ہے۔ جنسی تصورات ہماری پیشانیوں پر نہیں لکھے ہوتے ہیں۔
