Bagong session

Magsimula ng bagong session dito! Anonymous at confidential.

I-acess ang nakaraang session.

Pakilagay ang iyong session ID para mapuntahan ulit ang iyong session!

Sa tuwing magsisimula ka ng bagong season makakatanggap ka ng session ID. Makikita mo ang session ID sa kanang itaas. Pakikopya ang iyong session ID kung sakaling gusto mong gamitin ang mga credential sa ibang pagkakataon o kung gusto mong magpatuloy sa ibang oras. Kung nawala ang iyong ID hindi ka makakapag-log in ulit. Sa ganitong sitwasyon, kailangan mong ulitin ang session. Batay sa sagot mo posibleng eligible kang makipag-ugnayan sa isang therapist. Palaging alamin ang batas sa iyong lugar tungkol sa mandatoryong pagre-report. Ang pagpapatuloy ng online na session ng paggamot ay palaging isang opsyon.
